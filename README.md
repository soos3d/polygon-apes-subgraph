# polygon-apes-subgraph

 Simple subgraph to index transfers and token details from the Polygon Apes collection.

## Quick start

Install thr Graph CLI

```sh
yarn global add @graphprotocol/graph-cli
```

Install dependencies

```sh
yarn
```

Run codegen

```sh
graph codegen
```

Run build

```sh
graph build
```

Deploy the subgraph
